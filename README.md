# gsc-asm
gsc-asm, a gsc assembler/disassembler for IW5 (Call of Duty: Modern Warfare 3)

## Usage
``./gsc-asm.exe <options> <file>``

available options:
* ``-stdout``
* ``-disasm``
* ``-asm``
